Overview

A browser extension that adds/automates Eloqua features to enhance ease-of-access and reduce the time taken to get things done in Eloqua. Upon installing and enabling the extension, an Idea button appears on the bottom left of the screen (only on browser tabs running Eloqua Web App). Clicking on the button opens up a console where you can see a list of actions that you can perform through this extension. The extension also has a settings pane that allows you to choose the features that you want and you can disable the rest to make sure that Eloqua behaves just the way you want it to.

Features

Save Assets

You can choose to enable the extension to save currently editing asset (Email/LP/Segment/Campaign/Form) in the most intuitive way – pressing Ctrl+S.

Copy Assets

Although, there is an option to clone existing assets in Eloqua, it is very rigid. This extension allows you to select multiple assets and copy them by pressing Ctrl+C. You can then set the names of the to-be-created assets, navigate to the destination folder and create the copies by pressing Ctrl+V.
( Note: Assets are not cloned until you paste them with Ctrl+V )

Delete Assets

Much like copying assets, there is a limitation of deleting one asset at a time in Eloqua. You can select and delete multiple assets by pressing Shift+Del. Active assets can’t be deleted through this option even though they are selected. Moreover, to prevent accidental deletion of assets, the extension will ask for your confirmation once. This feature can be enabled or disabled under the settings pane.

Test Emails

Send test emails to yourself by opening the Email in the editor and pressing Ctrl+Alt+M (M for' Myself'). Test emails can be sent to others by using Ctrl+Alt+T. You can also send test emails from the Campaign Canvas by selecting the Email Step and using the same key combinations. Buttons for the same are available under the console.

Fieldmerge Lookup

Search Fieldmerges and view their syntaxes from any screen within Eloqua. Open up the console via the Idea button and select the Fieldmerge Lookup option or press Ctrl+Alt+F on the keyboard – a popup will appear. You can enter a keyword on the textbox that says “Enter search keyword here…” and press enter or click on the Search button on the right. Once the search is complete, the popup lists all matching fieldmerges below. A Filter textbox appears beside the Search button. This can be used to instantly filter the results. Select a listed Fieldmerge to view its details on the details pane at the bottom of the popup.

My Reports

You can quickly navigate to My Reports in Eloqua by pressing Ctrl+Alt+R. Although, a button is available on the console that does just the same, you are encouraged to use the shortcut.

View Dependencies

Asset dependencies (Emails, Landing Pages, Segments and Forms) can be viewed by the shortcut Ctrl+Alt+D.

Naming Generator

Emails, Campaigns, Forms, Landing Pages and Segments can be named using the Naming Generator by double-clicking the asset name in the design/editor view. You can also invoke the Naming Generator at any time using Ctrl+Alt+N or through the button on the console.

Search LP with Vanity URL

Sometimes you are required to search for a Landing Page and all you know is the Landing Page’s vanity url. The Search LP with Vanity URL allows you to do just that. It can be accessed through the console.

Campaign Canvas Search

Searching assets on the Campaign Canvas is now easier with this extension. You no longer have to look through every step to find the asset you want. Just enter the asset name or search keyword in the Search Box at the bottom right corner of your Campaign Canvas.

Fieldmerge Validation

The extension automatically validates all Fieldmerge tags present in the Email/LP code when you click on the Save button and highlights any invalid syntax. Assets are saved only when all syntax are correct.

Renaming Assets

Go ahead and press F2 to rename the assets (Emails/LPs/Segments/Forms/Campaign Canvases).

Blindform URL Generation

Every form’s Blindform URL can be generated by clicking on the Get Blind Form Link button from the left pane in the Form design view. This also keys in the Fieldmerge syntaxes that are set for each field.

All popups displayed by the extension can be dragged. So, you can place them anywhere on your screen to comfortably work with Eloqua. You can also drag the Catalyst Icon (Light bulb) to place it anywhere on the screen.
